WEB DEVELOPMENT AS AN INDUSTRY

Examples of dramastic transformation in communication and commerce led by web development include e-commerce.Online auction sites such as eBay have changed the way the consumers find and purchase goods and services.Online retailers such as Amazon.com and Buy.com (amoung many others) have transformed the shopping and bargain hunting experience for many consumers

PRACTICAL WEB DEVELOPMENT

Since the commercialization of the web,web development has been a growing
industry.The growth of this industry is being driven by business by business wishing to use
their website to sell products and services to customers.

There is open source software for web develoment like BerkeleyDB,GlassFish
LAMP(Linux),Apache,MySQL,PHP) stack and Perl/ Plack.This has kept the cost of
learning web development to a minimum.Another contributing factor to the
growth of the industry has been the rise of easy-to-use WYSIWYG web-develop-
ment software,such as abode Dreamweaver BlueGriffon and Microsoft Visual
Studio.Knowledege of Hyper Text Mark up Language (HTML) or of programming
languages is still required to use such software,but the basics can be learned and
implemented quickly with the help of help files,technical books,internet tutorials,
or face-to-face training.

An ever growing set of tools and technologies have helped developers build more
dynamic and interactive websites.Further,web developers now help to deliver
applications as webservices which were traditionallyonly available as applicat
ions on a desk-based computer. this has allowed for many oppurtunities to de-
cenetralize information and media distribution.Examples can be seen with the rise
of cloud services such as Adobe Creative cloud,Dropbox and Google Docs.These
web services allow users to interact with applications from manylocations
instead of being tied to specific workstation for their appliction enviroment.
